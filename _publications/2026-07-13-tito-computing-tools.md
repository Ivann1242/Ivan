We introduce **TITO_Explore**, a software package for representing and computing with Translation-Invariant Total Orders (TITOs). We define a canonical window notation for TITOs and design and implement algorithms for several computational tasks: normalizing the window notation into canonical form, computing the inversion set, comparing the weak order between two TITOs, and computing the join of two TITOs. The weak order comparison algorithm partitions inversion sets into disjoint subsets, reducing the comparison to evaluations of paired subsets; the join algorithm represents inversions with an edge-weighted directed graph and reformulates the join as a weighted path problem.

Authors: Yifan Jing, Emma Yicheng Lou, Shaotian Sun, Yuxuan Wu.
